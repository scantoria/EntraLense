## Physical Verification Checklist

### For Each Sampled Asset:

1. **Asset Identification**
   - [ ] Locate physical asset
   - [ ] Verify asset tag/serial number matches records
   - [ ] Take photo of asset tag/serial number

2. **Location Verification**
   - [ ] Confirm physical location matches recorded location
   - [ ] Document any location discrepancies

3. **Condition Assessment**
   - [ ] Good - Fully functional, no visible damage
   - [ ] Fair - Functional with minor issues/wear
   - [ ] Poor - Degraded condition, may need replacement
   - [ ] Non-functional - Requires repair or disposal

4. **Assignment Verification**
   - [ ] Verify current user/department
   - [ ] Confirm user acknowledgment of asset custody

5. **Documentation**
   - [ ] Note any discrepancies
   - [ ] Record special conditions or concerns

---

## System Verification

- [ ] Cross-reference physical audit with EntraLense asset database
- [ ] Verify warranty status in vendor portal
- [ ] Check maintenance/repair records
- [ ] Confirm depreciation calculations are accurate
- [ ] Review insurance coverage for high-value assets

---

## Discrepancy Types

| Type | Description | Action Required |
|------|-------------|-----------------|
| Missing | Asset not found at recorded location | Investigation |
| Unrecorded | Physical asset not in system | Add to inventory |
| Location | Asset found in different location | Update records |
| Assignment | Different user than recorded | Update records |
| Condition | Condition differs from recorded | Update and assess |
| Serial Mismatch | Serial number doesn't match | Investigate |

---

## Discrepancy Resolution

For each discrepancy found:

1. [ ] Document discrepancy details
2. [ ] Assign resolution responsibility
3. [ ] Set resolution deadline
4. [ ] Update EntraLense database
5. [ ] Verify correction

---

## Post-Audit Actions

- [ ] Generate discrepancy report
- [ ] Update asset records in EntraLense
- [ ] Process disposals/transfers
- [ ] Schedule follow-up for unresolved items
- [ ] Submit final audit report
- [ ] Schedule next audit (recommended: annually)
